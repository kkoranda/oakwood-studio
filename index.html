Software for the built environment.

Oakwood Studio builds purpose-built software for the industries that design, assess, and manage buildings and infrastructure.

What We Do

We specialize in custom software for facility assessment, property condition analysis, and building intelligence. Our tools handle the full lifecycle — from structured field data collection on tablets through analysis, reporting, and client-ready deliverables.

We've built platforms that have powered the assessment of thousands of facilities across public schools, federal buildings, military housing, commercial portfolios, and institutional campuses.

Capabilities

Field Data Collection

iPad-first mobile applications for structured facility inspections with offline capability, photo capture, configurable surveys, and real-time data validation.

Web Platforms

Centralized data management, analysis dashboards, portfolio views, and role-based access for teams operating at scale.

Report Generation

Templatized, client-ready report output with programmatic data population, photologs, cost tables, and collaborative editing.

Data & Integration

Cost library integration, custom data pipelines, API development, and system interoperability for assessment workflows.

Track Record

  • Maryland Public Schools — Data collection platform and FCA tooling for a statewide assessment of 1,439 schools completed in a single year.
  • Chicago Public Schools — Assessment tools for a recurring annual assessment of ~500 schools, ongoing for 6+ years. Still actively maintained and developed today.
  • Bureau of Indian Affairs — Custom assessment tooling for 150 federal schools in approximately 4 months.
  • HUD / RAD Assessments — Nearly 100 Rental Assistance Demonstration PCAs with custom tooling for housing assessments.
  • U.S. Air Force Housing — Custom assessment tooling for military housing facilities.
  • National Retail & University Portfolios — Custom tooling for large-scale facility inventory projects across Albertsons, Casey's, Aldi, Emory University, University of Illinois, and others.

About

Oakwood Studio is led by Kevin Koranda, who started his career conducting facility condition assessments and property condition assessments before moving into software development. Every tool we build is informed by firsthand experience walking the buildings, documenting the deficiencies, and delivering the reports.

Kevin also works as a software engineer developing CAD software for the stone fabrication industry — demonstrating the ability to deliver across complex, specialized technical domains.
